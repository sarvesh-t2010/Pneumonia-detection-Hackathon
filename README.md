## PNEOUMONIA DETECTION WITH MACHINE LEARNING!
- Open the program via the url, press the image upload button and then press anaylyze to get a result (the website doesn't show a change if you upload the image).
  - Use the files in the "test_data" to test the model 
- The model is trained on these datasets: https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia?resource=download, https://www.kaggle.com/datasets/yusufmurtaza01/chest-xray-pneumonia-balanced-dataset?resource=download&select=train
- The backend is a group of 2 seperate models trained on the above datasets, the models were created using google's teachable program
